# slot-web 🎰

TypeScript + Vite で作った、CloverPit風ローグライク・スロットゲーム。

## モード

タイトル下のトグルで2つのモードを切り替えられる（進行状況はそれぞれ別に保存され、行き来できる）:

- **⚔️ ローグライク**: ノルマ×期限×お守りのラン攻略モード（下記）
- **🎰 フリー**: ノルマなしで好きなだけ回せる従来モード。💰1,000スタートで、
  なくなったらチャージできる。お守り効果は適用されない

## 遊び方（ローグライクモード）

1. 所持クレジット **100** からスタート。各ラウンドには **💀 ノルマ** と **制限スピン回数（基本10回）** がある
2. スピンでクレジットを増やし、ノルマ額に届いたら「💀 ノルマを支払う」で支払ってラウンドクリア
   （スピンを使い切った時点でノルマに届いていれば自動で支払われる）
3. クリアすると **🛒 ショップ** が開き、残ったクレジットで **お守り** を買ってパワーアップ
4. 次のラウンドはノルマが **倍増**（150 → 300 → 600 → 1,200 → …）
5. スピンを使い切ってノルマ未達、またはクレジットが尽きたら **💀 GAME OVER** — 最初から

進行状況（ラウンド・お守り・クレジット）は localStorage に自動保存される。

### お守り（ショップでランダム3種が並ぶ）

| お守り | 効果 | 価格 |
| --- | --- | --- |
| 🍒 さくらんぼのお守り | 🍒の配当が2倍 | 50 |
| 🍋 レモンのお守り | 🍋の配当が2倍 | 50 |
| 🍇 フルーツバスケット | 🍇と🍊の配当が2倍 | 70 |
| 💎 宝石箱 | 💎と🔔の配当が2倍 | 90 |
| 7️⃣ ラッキーセブン | 7️⃣の配当が2倍 | 100 |
| 🍀 四つ葉のクローバー | すべての配当が1.5倍 | 150 |
| ⏳ 砂時計 | 毎ラウンドのスピン回数+3 | 80 |
| 🧲 磁石 | スキャッターの必要個数-1 | 90 |
| 🐷 ブタの貯金箱 | ハズレのときBETの半分が返ってくる | 70 |

同じお守りを複数持てば効果は重ねがけで累積する。

## スロットのルール

- **5リール×3行**。横・斜めのライン上なら**どの位置でも**、同じ絵柄が **3個以上並ぶ** と配当（BET × 倍率）
- ✨ **スキャッター**: 画面のどこでも 7️⃣ が2個以上、または 🍒 が5個以上で配当
- **BET** は −/＋で 1-2-5 刻み（10, 20, 50, 100, …）、上限なし。長押しで連続変更
- **AUTO** ボタンで自動スピン。もう一度押すか、クレジット不足・ラウンド終了・
  ゲームオーバー・モード切替などで自動停止する

### 配当表

| 絵柄 | 3個 | 4個 | 5個 |
| --- | --- | --- | --- |
| 7️⃣ | ×20 | ×100 | ×500 |
| 💎 | ×6 | ×30 | ×200 |
| 🔔 | ×4 | ×15 | ×100 |
| 🍇 | ×2 | ×10 | ×50 |
| 🍊 | ×2 | ×8 | ×30 |
| 🍋 | ×1 | ×4 | ×15 |
| 🍒 | ×1 | ×4 | ×10 |

### スキャッター配当（画面のどこでもOK）

| 絵柄 | 条件と倍率 |
| --- | --- |
| 7️⃣ | 2個 ×1 ／ 3個 ×5 ／ 4個 ×30 |
| 🍒 | 5個 ×1 ／ 6個 ×5 |

### 演出

- 🎆 **BIG WIN / MEGA WIN**: BETの15倍/50倍以上の当たりで、紙吹雪＋画面シェイク＋
  金額カウントアップの全画面演出
- ⚡ **リーチのピカッ**: 2個揃って「あと1つで当たり」になった瞬間、そのマスが光る
- 🔥 **7リーチ**: 7️⃣が2個見えていると最後のリールが溜まって赤く光る
- 🔊 **効果音**: WebAudioで合成（右上の🔊ボタンでミュート切替）

### 📊 統計（左上のボタン）

- **クレジット推移グラフ**: 直近300スピンの所持クレジットを折れ線で表示。
  ホバーでスピンごとのBET/WINを確認できる
- **実績**: 総スピン・当たり率・総BET/WIN・実績還元率（累計、localStorageに保存）
- **確率**: 現在のお守り効果込みの理論値をその場で4,000回シミュレーションして表示
  （絵柄ごとのライン当たり率・スキャッター率・理論還元率）

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック + 本番ビルド (dist/)
npm run preview  # ビルド結果のプレビュー
```

## 構成

- `src/game.ts` — リール定義・抽選・配当判定などの純粋ロジック（DOM 非依存）
- `src/charms.ts` — ローグライクモードのお守り・ノルマ定義
- `src/stats.ts` / `src/chart.ts` — プレイ実績の記録・確率シミュレーション・グラフ描画
- `src/main.ts` — UI 構築・リールアニメーション・ラウンド進行・状態管理
- `src/fx.ts` / `src/sound.ts` — 視覚演出と効果音
- `src/style.css` — 見た目
