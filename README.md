# slot-web 🎰

TypeScript + Vite で作ったブラウザで遊べるスロットゲーム。

## 遊び方

- **SPIN** ボタン（またはスペースキー）でリールを回す
- **BET** の −/＋ でベット額を変更。10 から 1-2-5 刻み
  （10, 20, 50, 100, 200, 500, …）で**上限なし**に上げられる。長押しで連続変更
- **5リール×5行**。横・斜めのライン上なら**どの位置でも**、
  同じ絵柄が **3個以上並ぶ** と配当（BET × 倍率）。
  1回のスピンで複数並べば全部もらえる
- ✨ **スキャッター**: ラインに揃わなくても、画面のどこでも 7️⃣ が3個以上、
  または 🍒 が8個以上あれば配当がもらえる
- クレジットは localStorage に保存され、なくなったらチャージできる

### 演出

- 🎆 **BIG WIN / MEGA WIN**: BETの15倍/50倍以上の当たりで、紙吹雪＋画面シェイク＋
  金額カウントアップの全画面演出
- 🔥 **リーチ**: 7️⃣が2個見えていると最後のリールが溜まって赤く光る
- 🔊 **効果音**: スピン・リール停止・当たり・ビッグウィンなどをWebAudioで合成
  （右上の🔊ボタンでミュート切替、設定は保存される）
- リール停止時のフラッシュ、当たりメッセージのポップなど細かい演出も追加

### 配当表

| 絵柄 | 3個 | 4個 | 5個 |
| --- | --- | --- | --- |
| 7️⃣ | ×10 | ×50 | ×250 |
| 💎 | ×3 | ×15 | ×100 |
| 🔔 | ×2 | ×8 | ×50 |
| 🍇 | ×1 | ×5 | ×25 |
| 🍊 | ×1 | ×4 | ×15 |
| 🍋 | ×1 | ×2 | ×8 |
| 🍒 | ×1 | ×2 | ×5 |

### スキャッター配当（画面のどこでもOK）

| 絵柄 | 条件と倍率 |
| --- | --- |
| 7️⃣ | 3個 ×1 ／ 4個 ×5 ／ 5個 ×30 |
| 🍒 | 8個 ×1 ／ 10個 ×5 |

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック + 本番ビルド (dist/)
npm run preview  # ビルド結果のプレビュー
```

## 構成

- `src/game.ts` — リール定義・抽選・配当判定などの純粋ロジック（DOM 非依存）
- `src/main.ts` — UI 構築・リールアニメーション・状態管理
- `src/style.css` — 見た目
